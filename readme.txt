Plugin to generate Image Sitemaps in XML-Format for your WordPress blog. 

== Description ==

Simple Plugin to generate a image sitemap for your WordPress-blog with all the image URLs that are used in your Blog. 

== Installation ==

Here's how you can install the plugin:

1. Upload the plugins folder to the /wp-content/plugins/ directory.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Expand the Tools menu from WordPress dashboard sidebar and select "Image Sitemap generate"
1. Click the "Generate Sitemap" button to create your XML Sitemap.
1. Than you can submit it to Google or other SE.
